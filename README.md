What is the difference between var, let, and const? 
var was once used in js to declare variables, because of hoisting now it's not used anymore. let is modern es6 variable declaration system which is reassignable. const is modern es6 variable declaration system which is not reassignable.

 What is the spread operator (...)?
 The spread operator (...) is used to expand an array, object, or iterable into individual elements.

 What is the difference between map(), filter(), and forEach()?
 map()-Used when you want to modify or transform every element and returns a new array. 
 filter()-Used when you want to keep only elements that meet a condition and returns a new array with matching elements.
 forEach()-Used when you want to perform an action for each element.
